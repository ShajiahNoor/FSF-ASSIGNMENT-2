# NEXORA Landing Page

A modern, responsive landing page designed with HTML5 and CSS3. The project showcases a visually appealing hero section with a background image, engaging call-to-action buttons, and a subscription form.

## Features

- Responsive Design
- Flexbox Layout
- Media Queries
- Modern Hero Section
- Call-to-Action Buttons
- Subscription Form
- Mobile Friendly
- Clean and Organized Code

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

## Author

Created as a responsive web design project to demonstrate modern front-end development techniques.
